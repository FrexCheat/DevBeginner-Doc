### 项目介绍

DevBeginner-Doc 是一个使用 [Next.js](https://nextjs.org/) 和 [Fumadocs](https://fumadocs.dev/) 构建的编程入门教程，力求描述详尽、易于理解的教程内容，专门为编程初学者设计，尤其针对计算机科学或软件工程专业的大学新生，他们通常具有有限的编程经验。

### Work with Claude Code

🌟 DevBeginner-Doc 支持 Claude Code 协作，提示词文件：[CLAUDE.md](https://github.com/FrexCheat/DevBeginner-Doc/blob/main/CLAUDE.md)

你可以在这个文件中找到一些具体的关于本项目的协作规范以及解释，包括但不限于：代码风格指南、文档内容要求、提交信息规范、其他相关的协作流程和工具使用说明

### 项目结构

```bash
DevBeginner-Doc/
├── .github/             # Github actions workflow files
├── app/                 # Next.js application files
├── components/          # Custom UI components which are used in pages or docs
├── content/docs/        # Documentation content (MDX files)
├── hooks/               # React/Next.js tsx files used for hook
├── lib/                 # Utility libraries
├── mdx-components.tsx/  # Register components for all mdx files of fumadocs
├── public/              # Public static sources such as logo and ico of the site
├── source.config.ts     # Fumadocs source config
└── styles/              # Styles files of the site
```

### 如何贡献

如果你想为 DevBeginner-Doc 做出贡献，可以通过以下步骤：

1. Fork 这个仓库
2. 创建一个新的分支
3. 在你的分支上进行修改
4. 提交你的修改并推送到你的 fork
5. 创建一个 Pull Request

我们欢迎任何形式的贡献，包括但不限于：

- 提交新的文档
- 修改现有文档
- 修复拼写和语法错误
- 提供反馈和建议

感谢你的贡献！我们期待与你一起改进 DevBeginner-Doc。
